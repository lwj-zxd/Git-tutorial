1. git init

##  记录每次更新到仓库

![](https://git-scm.com/book/en/v2/images/lifecycle.png)

```console
1. 检查当前文件状态
	git status
2. 跟踪新文件
	git add
3. 暂存已修改的文件
	git add
4. 状态简览
	git status -s
5. 忽略文件
	.gitignore
6. 查看已暂存和未暂存的修改
	git diff
7. 提交更新
	git commit
8. 跳过使用暂存区域
	git commit -a
9. 移除文件
	git rm
10. 移动文件
	git mv file_from file_to
```
